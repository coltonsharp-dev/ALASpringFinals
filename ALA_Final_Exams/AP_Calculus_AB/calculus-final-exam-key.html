Section A · Multiple Choice (1–29)
Question 1 Foundations · Limits
Evaluate:   limx→3 x² − 9x − 3
Solution
Direct sub gives 0/0 (indeterminate). Factor: x² − 9 = (x − 3)(x + 3). Cancel (x − 3):   limx→3 (x + 3) = 6.
Question 2 Foundations · Continuity
For f(x) = (x² − 4)/(x − 2), what type of discontinuity exists at x = 2?
Solution
x² − 4 = (x − 2)(x + 2). The (x − 2) cancels, so the limit at x = 2 exists and equals 4. But f(2) is undefined (denominator = 0). Limit exists but f is undefined → removable discontinuity (hole).
Question 3 Foundations · Limits at ∞
Evaluate:   limx→∞ 3x² + 5x − 16x² − 2x + 4
Solution
Top degree (2) = bottom degree (2) → limit = ratio of leading coefficients = 3/6 = 1/2.
Question 4 Foundations · Unit Circle
Find:   sin(7π/6)
Solution
7π/6 is in Quadrant III (where sine is negative). Reference angle = π/6. sin(π/6) = 1/2, so sin(7π/6) = −1/2.
Question 5 Theorems · IVT
A function f is continuous on [1, 5] with f(1) = −2 and f(5) = 8. By the Intermediate Value Theorem, which value of k is GUARANTEED to have a corresponding c in (1, 5) where f(c) = k?
Solution
IVT requires k to be between f(1) = −2 and f(5) = 8. Only k = 4 lies strictly between −2 and 8. The others (−5, −3, 9, 12) all fall outside [−2, 8].
Question 6 Theorems · MVT
A function f is differentiable on [2, 8] with f(2) = 4 and f(8) = 22. By the Mean Value Theorem, there exists c in (2, 8) such that f'(c) =
Solution
MVT guarantees: f'(c) = [f(8) − f(2)] / (8 − 2) = (22 − 4)/6 = 18/6 = 3.
Question 7 Differentiation · Power Rule
If f(x) = 4x³ − 2x² + 7x − 5, find f'(x).
Solution
Power rule term-by-term: 4·3x² − 2·2x + 7 − 0 = 12x² − 4x + 7. The constant −5 differentiates to 0.
Question 8 Differentiation · Product Rule
If f(x) = x²·sin(x), find f'(x).
Solution
Product rule: (fg)' = f'g + fg'. Here f = x², g = sin(x). So f' = 2x, g' = cos(x). Result: 2x sin(x) + x² cos(x).
Question 9 Differentiation · Quotient Rule
If f(x) = (3x + 1)/(x − 2), find f'(x).
Solution
Quotient rule: (low·d-high − high·d-low)/low². Numerator: 3(x − 2) − (3x + 1)(1) = 3x − 6 − 3x − 1 = −7. So f'(x) = −7/(x − 2)².
Question 10 Differentiation · Chain Rule
If f(x) = sin(3x²), find f'(x).
Solution
Chain rule: outside derivative · inside derivative. Outside: cos(u). Inside: u = 3x², u' = 6x. Result: cos(3x²) · 6x = 6x cos(3x²).
Question 11 Differentiation · Trig
What is the derivative of f(x) = tan(x)?
Solution
Standard trig derivative: d/dx[tan(x)] = sec²(x). Memory: derivatives of "co-" functions get a negative sign — tan does NOT, since it's not a co-function.
Question 12 Differentiation · Log
If f(x) = ln(5x² + 1), find f'(x).
Solution
d/dx[ln(u)] = u'/u (chain rule on ln). Here u = 5x² + 1, u' = 10x. Result: 10x/(5x² + 1).
Question 13 Implicit Differentiation
For the curve x² + y² = 25, find dy/dx.
Solution
Differentiate both sides w.r.t. x: 2x + 2y(dy/dx) = 0. Solve: dy/dx = −2x/(2y) = −x/y.
Question 14 Implicit · Product Term
For the curve xy + y² = 6, find dy/dx.
Solution
Differentiate using product rule on xy: y + x(dy/dx) + 2y(dy/dx) = 0. Group dy/dx terms: (x + 2y)(dy/dx) = −y. So dy/dx = −y/(x + 2y).
Question 15 App. of Derivatives · Extrema
For f(x) = x³ − 3x² − 9x + 5, at which x-value does a local minimum occur?
Solution
f'(x) = 3x² − 6x − 9 = 3(x − 3)(x + 1). Critical points: x = −1 and x = 3. Second derivative test: f''(x) = 6x − 6. f''(−1) = −12 < 0 (local max). f''(3) = 12 > 0 (local min). So local min at x = 3.
Question 16 App. of Derivatives · L'Hôpital
Evaluate:   limx→0 sin(2x)x
Solution
Direct sub gives 0/0 → indeterminate. Apply L'Hôpital: limx→0 [2cos(2x) / 1] = 2cos(0) = 2.
Question 17 App. of Derivatives · Related Rates
A spherical balloon's radius is increasing at 2 cm/s. When r = 5 cm, how fast is the volume changing?  [V = (4/3)πr³]
Solution
Differentiate V w.r.t. t: dV/dt = 4πr² · (dr/dt). Substitute r = 5, dr/dt = 2: dV/dt = 4π(25)(2) = 200π cm³/s.
Question 18 App. of Derivatives · Concavity
For f(x) = x⁴ − 4x³, on which interval(s) is the graph concave up?
Solution
f'(x) = 4x³ − 12x², f''(x) = 12x² − 24x = 12x(x − 2). f'' > 0 (concave up) when x < 0 OR x > 2. So concave up on (−∞, 0) ∪ (2, ∞).
Question 19 Antiderivatives · +C Rule
Find:   ∫(6x² − 4x + 3) dx
Solution
Reverse power rule term-by-term (add 1 to exponent, divide by new exponent): 6x²→2x³, −4x→−2x², 3→3x. Indefinite integral REQUIRES +C: 2x³ − 2x² + 3x + C.
Question 20 Antiderivatives · Trig
Find:   ∫sec²(x) dx
Solution
Since d/dx[tan(x)] = sec²(x), the antiderivative is tan(x) + C. (Antidifferentiation reverses differentiation.)
Question 21 U-Substitution
Find:   ∫2x · cos(x²) dx
Solution
Let u = x², du = 2x dx. The integral becomes ∫cos(u) du = sin(u) + C. Replace: sin(x²) + C.
Question 22 U-Sub · Definite
Evaluate:   ∫02 3x² · ex³ dx
Solution
u = x³, du = 3x² dx. Change bounds: x = 0 → u = 0,   x = 2 → u = 8. New integral: ∫08 eu du = e⁸ − e⁰ = e⁸ − 1.
Question 23 Riemann Sum · Left
Approximate ∫04 f(x) dx using a LEFT Riemann sum with n = 4 subintervals (Δx = 1) using the table:
x01234
f(x)5781012
Solution
Left endpoints: f(0), f(1), f(2), f(3) = 5, 7, 8, 10. Sum = (5 + 7 + 8 + 10)·Δx = 30·1 = 30. (Skip the rightmost f(4) = 12 — left endpoints don't include the right edge.)
Question 24 Riemann · Over/Under
For an increasing and concave up function, a LEFT Riemann sum produces an estimate that is:
Solution
For an increasing function, left rectangles use the smaller (leftmost) value on each subinterval, so the rectangles sit BELOW the curve. Result: underestimate. (Concavity affects trapezoid/midpoint, not left/right for monotonic functions.)
Question 25 Definite Integrals · Properties
Given that ∫03 f(x) dx = 7 and ∫35 f(x) dx = −2, find ∫05 f(x) dx.
Solution
Integral additivity: ∫05 = ∫03 + ∫35 = 7 + (−2) = 5.
Question 26 Definite Integrals · Graphs
The graph of f consists of a semicircle of radius 3 above the x-axis from x = −3 to x = 3, plus a triangle below the x-axis with base 4 and height 2 from x = 3 to x = 7. Find ∫−37 f(x) dx.
Solution
Semicircle area: (1/2)π(3)² = 9π/2 (above → POSITIVE). Triangle area: (1/2)(4)(2) = 4 (below → NEGATIVE). Net signed area: 9π/2 − 4.
Question 27 FTC Part 1 · Chain Rule
If g(x) = ∫0x² sin(t) dt, find g'(x).
Solution
FTC Part 1 with chain rule: replace t with the upper bound, then multiply by the derivative of the upper bound. g'(x) = sin(x²) · d/dx(x²) = sin(x²) · 2x = 2x sin(x²).
Question 28 App. of Integrals · Area
Find the area of the region bounded by y = 2x and y = x² from x = 0 to x = 2.
Solution
On [0, 2]: 2x ≥ x² (test x = 1: 2 ≥ 1 ✓). Area = ∫02(2x − x²) dx = [x² − x³/3]02 = (4 − 8/3) − 0 = 12/3 − 8/3 = 4/3.
Question 29 Physics · SHM
A particle moves along the x-axis with position x(t) = 4cos(2t). What is its velocity at t = π/4?
Solution
v(t) = x'(t) = −8 sin(2t) (chain rule: derivative of cos is −sin, times 2). At t = π/4: v(π/4) = −8 sin(π/2) = −8 · 1 = −8.
Section B · Multi-Part Numerical (30)
Question 30 Multi-Part · Particle Motion
A particle moves along the x-axis with velocity v(t) = 3t² − 12t + 9  for  0 ≤ t ≤ 5.

Answer all four parts. Enter exact numerical values (no units).
Part (a) The smaller value of t at which the particle is at rest:
1
Part (b) The larger value of t at which the particle is at rest:
3
Part (c) The total displacement from t = 0 to t = 5:  ∫05 v(t) dt
20
Part (d) The acceleration at t = 2:  a(2)
0
Worked Solution
(a) & (b) Particle at rest:  Set v(t) = 0.  3t² − 12t + 9 = 3(t² − 4t + 3) = 3(t − 1)(t − 3) = 0.  So t = 1 and t = 3.

(c) Displacement:  ∫05(3t² − 12t + 9) dt = [t³ − 6t² + 9t]05 = (125 − 150 + 45) − 0 = 20.

(d) Acceleration at t = 2:  a(t) = v'(t) = 6t − 12.  a(2) = 12 − 12 = 0.  (Note: this is when the particle's velocity reaches its minimum on the interval.)
Section C · Free Response (31–35)
Question 31 FRQ · Related Rates
A 13-foot ladder leans against a vertical wall. The bottom of the ladder is sliding away from the wall at a rate of 2 ft/s.

(a) How fast is the top of the ladder sliding down the wall when the bottom is 5 ft from the wall?

(b) Justify why your answer to (a) must be negative, in context.
(a) Setup & Differentiate
Let x = horizontal distance from wall, y = height of top, with x² + y² = 13² = 169.
Differentiate w.r.t. t:  2x(dx/dt) + 2y(dy/dt) = 0.

When x = 5: y² = 169 − 25 = 144, so y = 12.
Given dx/dt = 2 ft/s. Solve for dy/dt:
2(5)(2) + 2(12)(dy/dt) = 0 → 20 + 24(dy/dt) = 0 → dy/dt = −5/6 ft/s.
(b) Justification
The top of the ladder is moving DOWN, so its height y is DECREASING. A decreasing y means dy/dt < 0. The negative sign in −5/6 ft/s communicates that the height is decreasing at a rate of 5/6 ft per second.
Question 32 FRQ · MVT Justification
A function f is continuous on [2, 10] and differentiable on (2, 10), with the values:
t24710
f(t)591421
(a) Use the Mean Value Theorem to justify why there must exist a value c in (2, 10) such that f'(c) = 2.

(b) Use a left Riemann sum with the 3 subintervals shown to approximate ∫210 f(t) dt.
(a) MVT Citation
Average rate of change on [2, 10]:
[f(10) − f(2)] / (10 − 2) = (21 − 5)/8 = 16/8 = 2.

"Since f is continuous on [2, 10] and differentiable on (2, 10), by the Mean Value Theorem there exists at least one value c in (2, 10) such that f'(c) = [f(10) − f(2)] / (10 − 2) = 2."

This is the exact citation template — both conditions stated, full formula written, conclusion in interval notation.
(b) Left Riemann Sum
Subintervals: [2,4], [4,7], [7,10] with widths 2, 3, 3. Use LEFT endpoints: f(2), f(4), f(7).
Sum = 2·f(2) + 3·f(4) + 3·f(7) = 2(5) + 3(9) + 3(14) = 10 + 27 + 42 = 79.
Question 33 FRQ · Area & Volume
Let R be the region in the first quadrant bounded by y = √x and y = x/2.

(a) Find the x-coordinates of the points where the two curves intersect.

(b) Find the area of region R.

(c) Find the volume of the solid generated when R is rotated about the x-axis.
(a) Intersection
Set √x = x/2. Square: x = x²/4 → 4x = x² → x² − 4x = 0 → x(x − 4) = 0.
Intersections at x = 0 and x = 4.
(b) Area
On [0, 4]: top = √x, bottom = x/2 (test x = 1: √1 = 1, 1/2 = 0.5 ✓).
Area = ∫04(√x − x/2) dx = [⅔x3/2 − x²/4]04
= (⅔·8 − 4) − 0 = 16/3 − 12/3 = 4/3.
(c) Volume (Washer Method)
V = π∫04[(Router)² − (Rinner)²] dx = π∫04[(√x)² − (x/2)²] dx
= π∫04(x − x²/4) dx = π[x²/2 − x³/12]04
= π[8 − 64/12] = π[8 − 16/3] = π[24/3 − 16/3] = 8π/3.
Question 34 FRQ · Graph of f'
The graph of f'(x), the derivative of f, on [0, 8] consists of two pieces:
  • • A semicircle of radius 2 above the x-axis from x = 0 to x = 4
  • • A line segment from (4, 0) to (8, −4)
It is given that f(0) = 5.

(a) Find all x-values where f has a local maximum on (0, 8). Justify.

(b) Find f(4).

(c) On what interval(s) is f concave down on (0, 8)? Justify.
(a) Local Maximum
f has a local max where f' changes from POSITIVE to NEGATIVE.
On [0, 4], f' (the semicircle) is positive. At x = 4, f' = 0. After x = 4, f' is negative (line dips below x-axis).
f' goes + → − at x = 4. So f has a local maximum at x = 4.
(b) f(4)
By FTC:  f(4) = f(0) + ∫04 f'(x) dx.
∫04 f'(x) dx = area of semicircle = (1/2)π(2)² = 2π.
So f(4) = 5 + 2π.
(c) Concave Down
f is concave down where f' is DECREASING.
Semicircle: f' increases on (0, 2), decreases on (2, 4).
Line from (4, 0) to (8, −4): slope = −1, so f' is decreasing on (4, 8).
f is concave down on (2, 8).
Question 35 FRQ · Table & Riemann
The velocity of a car (in ft/s) is recorded at 5-second intervals:
t (sec)05101520
v(t)3045505540
(a) Use a midpoint Riemann sum with 2 equal subintervals to approximate ∫020 v(t) dt.

(b) Interpret ∫020 v(t) dt in context, with units.

(c) Use a trapezoidal sum with the 4 subintervals shown to approximate ∫020 v(t) dt.
(a) Midpoint Sum (n = 2)
Two equal subintervals: [0, 10] and [10, 20], each of width 10. Midpoints: t = 5 and t = 15.
Sum = 10·v(5) + 10·v(15) = 10(45) + 10(55) = 450 + 550 = 1000 ft.
(b) Interpretation
"∫020 v(t) dt represents the total distance, in feet, that the car travels from time t = 0 to t = 20 seconds."

Required for full credit: what the integral represents, units (feet), AND the time interval (0 to 20 seconds).
(c) Trapezoidal Sum (n = 4)
Width of each subinterval: Δt = 5.
Trap = (Δt/2)[v(0) + 2v(5) + 2v(10) + 2v(15) + v(20)]
= (5/2)[30 + 2(45) + 2(50) + 2(55) + 40]
= (5/2)[30 + 90 + 100 + 110 + 40]
= (5/2)(370) = 925 ft.
